# 카카오톡 선물하기 ERD (Entity-Relationship Diagram)

M0 기획 설계 단계에서 도출된 핵심 데이터베이스 테이블 관계도입니다. 
팀 가이드라인의 제약조건(외부 결제 모듈 및 주소 API 미연동)을 반영하여 주문(orders) 테이블 내에 상태 제어 컬럼을 두어 Mock 처리가 가능하도록 설계했습니다.

## 📊 ERD 다이어그램

![카카오 shop ERD](./erd.png)

## 🗂️ 주요 테이블 설명
* **`users`**: 서비스 이용자 정보 (카카오 계정 연동 기준)
* **`products`**: 선물하기 홈 및 상세페이지에 노출될 상품 정보
* **`orders`**: 구매 및 선물 내역 (가짜 결제 상태 및 선물 사용 여부 포함)
